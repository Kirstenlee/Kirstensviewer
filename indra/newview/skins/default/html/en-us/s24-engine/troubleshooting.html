Troubleshooting Guide

Home Common Settings Troubleshooting
Common issues and solutions for S24 Engine settings. If problems persist, use RESETS → Factory Reset to restore all defaults.

Reflections Not Working

Symptom: Reflection probe tweaks have no visible effect, or reflections look wrong after enabling/disabling.
Solution:
  1. Go to RESETS tab
  2. Click Purge Shader Cache
  3. Restart viewer
  4. Verify Preferences → Graphics → Advanced → Reflections is enabled

Low FPS / Poor Performance

Symptom: Frame rate drops significantly after adjusting settings.
Solution:
  • Reduce Shadow Resolution Scale: Lower from 3.0 to 1.5-2.0
  • Lower Draw Distance: Set RenderFarClip to 128-192 (LL default is 256)
  • Disable SSR: Screen-space reflections are GPU-intensive
  • Reduce LOD Factors: Lower RenderVolumeLODFactor to 2.0-3.0 (S24 default 4.125)
  • Verify Shadow Throttling: Should be enabled by default

Stuttering / Hitching

Symptom: Periodic freezes or stutters during movement.
Solution:
  • Reduce Render Thread Count: Too many threads can cause contention (check auto-detected value in CPU tab)
  • Check VRAM Budget: 0 = auto-detect is usually best, manual override rarely needed
  • Purge Caches: Go to RESETS tab, purge inventory and texture caches
  • Texture Memory: Viewer manages this dynamically - manual changes rarely help

Water Looks Wrong

Symptom: Water appears too shiny, too dark, or has color issues.
Solution:
  • Reset Water Settings: Metallic=1.0 (LL default), Roughness Override=0.0 (use environment)
  • Disable Custom Tint: Uncheck "Enable Custom Tint" in Water tab if enabled
  • Check Reflections: Reflection Probes must be enabled in Preferences → Graphics → Advanced
  • Purge Shaders: Water shader changes may require cache purge + restart

Shadow Acne / Light Leaks

Symptom: Surfaces show speckled shadow patterns or light bleeding through geometry.
Solution:
  • Adjust Shadow Bias: Multiple bias settings exist (spot, deferred, tree) - see Shadows tab for specifics
  • Check Resolution Scale: LL default is 3.0 - higher values may show more artifacts
  • Normal Offset: Increase slightly if shadow acne persists after bias adjustment

Smooth Movement Feels Wrong

Symptom: Movement feels sluggish, unresponsive, or "floaty" after enabling smooth keyboard movement.
Solution:
  • Reduce Acceleration: Lower from 0.2 to 0.1 seconds for snappier response (S24 defaults)
  • Reduce Deceleration: Lower from 0.15 to 0.05 seconds for quicker stops
  • Disable if Unnecessary: Feature is purely aesthetic - uncheck if you prefer instant response

Settings Reset After Restart

Symptom: KVTweaks settings don't persist between sessions.
Solution:
  • Check Permissions: Ensure settings file (settings.xml) is not read-only
  • Exit Properly: Use Quit option, not force-close
  • Verify Save: Most settings save immediately when changed

Crashes After Changes

Symptom: Viewer crashes or becomes unstable after adjusting settings.
Solution:
  1. Start viewer with -noprobe command line flag (disables reflection probes)
  2. Open KVTweaks → RESETS tab
  3. Click Factory Reset
  4. Click Purge All Caches
  5. Restart normally
  6. Re-enable settings gradually to isolate problematic setting

Nuclear Option: Complete Reset

When All Else Fails: Return to absolute default state.
Full Reset Procedure:
  1. RESETS → Factory Reset (restores all KVTweaks defaults)
  2. RESETS → Purge All Caches (clears shader, inventory, texture caches)
  3. Preferences → Graphics → Reset to Defaults (main graphics settings)
  4. Restart viewer
  5. Verify standard LL settings work before re-tweaking
Still Having Issues? Document your exact settings (screenshot each KVTweaks tab) and system specs (GPU, RAM, driver version) before seeking help. Most issues are GPU driver related - try updating drivers first.
