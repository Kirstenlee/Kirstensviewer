Quick Start

  • Most settings take effect immediately - no restart needed
  • Shader changes (Reflections enable/disable) require: RESETS → Purge Shader Cache → Restart
  • Hover over controls in KVTweaks for detailed tooltips
  • Use RESETS tab to restore defaults or purge caches

Tab Overview

Tab Purpose
CPU & Memory Core count, thread pool, RAM allocation, VRAM budget
Rendering LOD factors, draw distance, performance tuning
SSR Screen-space reflections (global, for all surfaces)
Water Water appearance: metallic, roughness, color, reflections
Shadows Shadow quality, bias, throttling, resolution

Important Notes

  • These are expert controls - for experienced users
  • Changes can impact stability and performance
  • TPV Policy compliant - all settings can be reset to Linden Lab defaults
  • Reflections/Water enhancements are opt-in and default to standard values
  • Movement smoothing is disabled by default
