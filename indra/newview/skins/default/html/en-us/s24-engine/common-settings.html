Common Settings & Defaults

Home Common Settings Troubleshooting
This page lists the most frequently adjusted settings and their default values. All defaults match Linden Lab standards unless noted as S24-specific enhancements.

CPU & Memory

Setting Default Notes
Render Thread Count Auto-detected KVdefault_cores sets initial value, viewer auto-configures based on CPU
Max VRAM Budget 0 (Auto) 0 = auto-detect GPU memory at startup, then dynamically managed by viewer. Manual override available but not recommended.
Texture Memory (MB) Auto-managed Viewer adjusts based on VRAM budget and scene complexity. Not a fixed allocation.

Rendering

Setting Default Notes
RenderVolumeLODFactor 4.125 S24 default - controls prim detail level (higher = more detail)
MeshLODBoost 0 Additional mesh quality boost (0-2 range, 0 = LL default)
RenderFarClip 256 Draw distance in meters (LL default, typical range 96-512)

SSR (Screen-Space Reflections)

Setting Default Notes
SSR Enable Disabled (false) LL feature - off by default, GPU-intensive when enabled
SSR Iterations 9 Ray march steps (LL default, higher = better quality, lower FPS)
SSR Ray Step 1.0 Distance per ray step (LL default)

Water

Setting Default Notes
Water Metallic 1.0 S24 runtime control - LL hardcoded to 1.0 (fully metallic mirror)
Water Roughness Override 0.0 0.0 = use environment blur (LL behavior), 0.01-1.0 = manual override
Water Fog Density Environment-controlled Set by water environment settings (no global default)

Shadows

Setting Default Notes
RenderShadowResolutionScale 3.0 S24 default - higher than LL due to advanced cascade system (LL typically uses lower values)
Shadow Bias Per-shader defaults Multiple bias settings exist (terrain, spot, deferred) - see Shadows tab
Shadow Throttle Enabled true LL default - reduces shadow updates when FPS drops

Reflections (Probes)

Setting Default Notes
Enable Reflection Probes User's Graphics Setting Enabled/disabled in Preferences → Graphics → Advanced. KVTweaks requires it enabled.
Probe Intensity 1.0 S24 runtime control - reflection strength multiplier (0.0-2.0+)
Probe Blur LOD Bias 0.0 LL default - adjusts reflection blur/sharpness
Ambient Multiplier 1.0 S24 runtime control - ambient probe contribution (0.0-2.0+)

Movement (Smooth Keyboard)

Setting Default Notes
Smooth Movement Enable Disabled (false) S24 feature - off by default, purely aesthetic enhancement
Acceleration Time 0.2 Seconds to reach full speed (S24 default)
Deceleration Time 0.15 Seconds to stop (S24 default)

Experimental (Selection Beam)

Setting Default Notes
Beam Style 2 (Hybrid) S24 default - 0=Particles Only, 1=Line Only, 2=Line+Glow (Lightsaber)
Line Width 3.0 S24 default - thickness in pixels (1.0-8.0, driver may clamp)
Custom Color Disabled (false) Use white (LL default) or custom RGB color
Pro Tip: Defaults shown are from settings.xml unless noted as "S24 default" or "LL default". Many settings are dynamically managed by the viewer (VRAM, texture memory, thread counts). Start with stock values and adjust incrementally while monitoring FPS.
